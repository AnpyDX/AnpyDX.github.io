Gosper's Hack 算法 ​

简介 ​

Gosper's Hack 是一个利用位运算，生成 n 元集合所有 k 元子集的算法，常常作为组合枚举类算法题的非递归解法。 该算法的时间复杂度与组合数一致。

其 C++ 实现如下：

cpp
void GospersHack(int n, int k) {
    int cur = (1 << k) - 1;
    int limit = cur << (n - k);

    while (cur <= limit) {
        int lb = cur & -cur;
        int r = cur + lb;
        cur = ((r ^ cur) >> __builtin_ctz(lb) + 2) | r;
        // 非 gcc 编译器可以用下方代码代替
        // cur = (((r ^ cur) >> 2) / lb) | r;
    }
}

将 GospersHack(5, 3) 的结果以二进制形式依次打印，有：

00111
01011
01101
01110
10011
10101
10110
11001
11010
11100

如果以 1 表示选中，0 表示未选中，可以发现结果正是 C53 表示的所有情况。

原理 ​

思想 ​

通过观察打印结果，我们可以发现二进制串 Ai→Ai+1 有这样的规律：将最后（右）一个 01 变为 10，同时将其右侧的 1 全部“堆放”到末尾。 例如：01110→10011。

实际上，这正是 Gosper's Hack 的核心：根据 n 和 k 得到第一个二进制串，再通过上述的规律依次得到所有组合！

但为什么这样做是正确的？一个简单的解释就是：

如果用二进制串来表示元素的选择情况，列举 n 元集合所有 k 元子集， 实际上就是找到 0000…⏟n 到 1111…⏟n 中所有 1 个数为 k 的二进制串。 而上述步骤保证了在 1 个数不变的情况下，Ai 与 Ai+1 间差值最小，从而确保不会遗漏组合。

更加详细的解释可以参见这篇 文章。

实现 ​

实现的核心在于如何实现以下两个操作：

  • 将最后一个 01 变为 10。
  • 将最后一个 01 右侧的 1 全部“堆放”到末尾。

接下来我们逐行分析实现代码。

由于组合是按照从小到大的方式生成的，第一个和最后一个组合可以通过下方代码得出。

cpp
int cur = (1 << k) - 1;
int limit = cur << (n - k);

以 GospersHack(5, 3) 为例，其中 cur = 00111， limit = 11100。

int lb = cur & -cur 计算出 cur 的 lowbit，也就是仅保留最右一位 1，其余全改写为 0 的二进制串。 例如，10110 的 lowbit 为 00010。具体原理详细参见 文章。

int r = cur + lb 计算出最后一个 01 进位后的结果。为什么是这样呢？

需要注意到，最后一个 01 的右侧一定满足这样的 形式 : …01111…⏟a000…⏟b （a 和 b 可以为 0）。

那么 cur 加上 lowbit 就能将最后一个 01 变为 10，即 r 可以表示为 …10000…⏟a+b。

至此，第一步操作已经完成，并且结果被存储在 r 中。

接下来我们来分析最为巧妙的最后一行代码，它将完成第二步操作。

cpp
cur = ((r ^ cur) >> __builtin_ctz(lb) + 2) | r;

r ^ cur 得到的是将 cur 中最后一个 01 转换为 11，并且抛弃其左侧内容后的结果。例如，10110→01110。

还记得最后一个 01 的右侧一定满足 形式 : …01111…⏟a000…⏟b 吗？

现在 r ^ cur 可以表示成 000…11111…⏟a000…⏟b。

注意到了吗？此时只要将这个二进制串右移 b+2 个单位就是对 cur 完成第二步后的结果！这正是 >> __builtin_ctz(lb) + 2) 所做的。

最后，由于先前 r ^ cur 已经抛弃了 01 左侧的内容，并且 r 中 10 的右侧全是 0，我们此时只需要将两个部分进行或运算 cur = (...) | r， 即可得到下一个组合！

应用 ​

93. 递归实现组合型枚举

题目描述 ​

从 1∼n 这 n 个整数中随机选出 m 个，输出所有可能的选择方案。

输入格式 ​

两个整数 n,m，在同一行用空格隔开。

输出格式 ​

按照从小到大的顺序输出所有方案，每行 1 个。

首先，同一行内的数升序排列，相邻两个数用一个空格隔开。

其次，对于两个不同的行，对应下标的数一一比较，字典序较小的排在前面（例如 1 3 5 7 排在 1 3 6 8 前面）。

输入样例 ​

5 3

输出样例 ​

1 2 3 
1 2 4 
1 2 5 
1 3 4 
1 3 5 
1 4 5 
2 3 4 
2 3 5 
2 4 5 
3 4 5

使用 Gosper's Hack 的非递归解法 ​

cpp
#include <iostream>
using namespace std;

int main() {
    int n, k;
    cin >> n >> k;
    
    k = n - k;
    
    int cur = (1 << k) - 1;
    int limit = cur << (n - k);
    
    while (cur <= limit) {
        for (int i = 1; i <= n; i++) {
            if (!(cur & (1 << n - i)))
                cout << i << " ";
        }
        cout << endl;
        
        int lb = cur & -cur;
        int r = cur + lb;
        cur = ((r ^ cur) >> __builtin_ctz(lb) + 2) | r;
    }
}
