# ad.BLOG

此仓库为个人博客源码，基于 [astro-erudite](https://github.com/jktrn/astro-erudite) 修改。

博客支持的 **Markdown** 控件请参考 `samples` 下由 [astro-erudite](https://github.com/jktrn/astro-erudite) 提供的样例。

## 可用 CLI

- 启动开发服务器：`npm run dev`
- 生成生产版本文件：`npm run build`
- 预览生产版本站点：`npm run preview`

> 注意：搜索功能仅在生产版站点可用
